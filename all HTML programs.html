►DEFINITIONS YOU MUST KNOW:

1.Webpage: A webpage is a document that can contain text, graphics, sound or video and can be viewed in a web browser.

2. Website: A website is a collection of interrelated web pages.

3. Web Browser: Web browsers are application software used to view web pages.

4. Web Server: A web server is the main computer which stores data, web sites etc.

5. Home Page: The first/starting page of any website which displays the details of the website.

6. Protocol: Protocol refers to the set of rules that computers use to communicate with each other.

7. HTML: The HTML tag represents the root of an HTML document. The HTML tag is the Container tag for all the other HTML elements.

8. HEAD: The HEAD tag is a container tag. It is the first element inside the HTML tag. The HEAD tag can include a title for the document.

9. TITLE: The TITLE tag is used inside the HEAD tag to mention the document title. The title appears on the top of the webpage i.e, in the Title Bar. It is a container tag

10. BODY: The BODY tag defines the document’s body. The BODY tag contains all the contents of an HTML document, such as text, hyperlinks, images, tables and lists etc. It is a container tag.

11. P: P tag is used to start a new paragraph. It is a container tag.

12. Hn: It is used to define the different heading levels. There are six heading levels defined with the H1 to H6 tags. h1 displays the text in the largest size and H6 in the smallest size.

13. BR: BR is an empty tag. It does not contain closing tag. This tag breaks the line and displays the remaining text from the next line, without leaving any space. It can also be used to insert blank lines.

14. HR: It is an empty tag. HR tag is used to draw a horizontal line between the block of text.

15. FONT: This tag is used to define the color, size and face of text in a web document. It is a container tag. FONT tag cannot be used without its attributes.

16. CENTER: CENTER tag is used to align the enclosed text to the center of the web page. It is a container tag.

17. UNORDERED LIST: An unordered list is a collection of related items that have no special order. Each item in the list is marked with a bullet. This list is created by using OL tag. It is a container tag.

18. ORDERED LIST: An ordered list is a collection of related items that are in sequential order.
i.e., 1,2,3…., A,B,C….., a,b,c…., I,ii,iii….., I,II,III,…… This list is created by using OL tag. It is a container tag.

19. LIST: Every item in the unordered list is marked using LI tag.

20. B, I, U: These tags are used to make the enclosed text bold face, underline and italic face. These are container tags.

21. IMG: It is used to insert images. It has no content and therefore does not need a closing tag. IMG tag is an empty tag.
